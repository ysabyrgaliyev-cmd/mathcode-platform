About MathCode

MathCode is an integrated K-12 curriculum and learning platform that teaches mathematics and computer science as a single, coherent subject. The methodology was developed at TAS&BA LLC (Katy, Texas) to address two well-documented U.S. educational gaps: declining mathematical proficiency and uneven access to computer-science instruction.

The methodology

MathCode is built on three principles:

Math first, syntax second. Coding constructs are introduced only when they make a mathematical idea visible. A loop is taught when it replaces repeated addition; a conditional is taught when it formalizes a rule; a function is taught when it captures a mapping.

Concrete → Representational → Abstract. Every project moves learners along the CRA progression, manipulating concrete objects, then visual representations, then symbolic code that drives those representations.

Applied projects, not exercises. Each unit culminates in a self-contained interactive project students can show, share, and extend.

Three tiers, twelve flagship projects

Tier 1 (K-3). Number sense, fractions, geometry, patterns using Scratch-style block coding.

Tier 2 (4-7). Ratios, algebra, coordinate geometry, probability using a side-by-side block ↔ Python view that bridges to text.

Tier 3 (8-12). Functions, trigonometry, statistics, calculus using Python with NumPy and Matplotlib for modeling and analysis.

Standards alignment

Every unit is mapped to Common Core State Standards for Mathematics (CCSS-M), the CSTA K-12 Computer Science Standards, and the K-12 Computer Science Framework. Detailed alignments are included in the per-tier curriculum guides.

Built by TAS&BA LLC

TAS&BA LLC was founded in Katy, Texas to deliver MathCode instruction directly to students and to disseminate the methodology to other educators. The platform is one component of a broader effort that includes published curriculum guides, classroom pilots, and partnerships with schools.
