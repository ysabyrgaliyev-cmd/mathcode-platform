Section 2 of 5

📦A magic box called a variable

Imagine you have a magic box. You can put anything inside — a number, a name, even a banana count. The box has a name on it, like "bananas" or "age". Whenever you want to know what's inside, you just say the box's name.

📦

In Python, a magic box is called a variable. Here's how I make one for my bananas:

bananas = 0

That little line says: "Make a box called bananas and put 0 inside." The = sign is not "equals" like in math. It means "put this in here."

bananasThe name of the box. You can pick any name you want!
=Means "put this in the box." Not "equals."
0What goes inside the box. We start at zero.

Once the box exists, I can tell the computer to show me what's inside. We use a special command called print:

print("Mango has", bananas, "bananas!")

Section 3 of 5

➕Now let's do some math

A box that holds 0 forever is a sad box. Let's add some bananas! In Python, we change a variable like this:

bananas = bananas + 1

Read it slowly: "Take what's in bananas, add 1, and put it back in the box." Each time we run that line, the box gets one more banana.

Tiny mind-bender

In math class, x = x + 1 would be silly — no number equals itself plus one. But in code, = means "put this in here," not "equals." So this works perfectly!

Now you try. The yellow box is empty — type a 1 in it (just press the number key). The Run button will wake up. Click it, and watch a banana fly to me. Click again to add another. And another.

Section 4 of 5

🔁Doing things many times

Clicking Run five times to get five bananas works. But what if I wanted one hundred bananas? My finger would fall off!

Computers are good at doing the same thing over and over. We just have to ask them. The magic word is for:

for i in range(5):
    bananas = bananas + 1

Read it like this: "For each time, 5 times in a row, add one banana." That little space at the start of line 2 (we call it indentation) tells Python: "this line is what gets repeated."

for i in range(5):"Do this 5 times." The i is just a counter — 0, 1, 2, 3, 4.
bananas = ...The 4 spaces at the front say "I belong inside the loop."

Now you tell the computer how many times to repeat. Type a number from 1 to 10 in the yellow box. (Try 7!)
