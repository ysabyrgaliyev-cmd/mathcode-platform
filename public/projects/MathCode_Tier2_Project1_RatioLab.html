How ratios work in code: A recipe for 4 servings uses specific amounts. To scale it, we calculate a ratio: ratio = new_servings / original_servings. Then multiply every ingredient by that ratio. This is proportional reasoning — when one quantity changes, all related quantities change by the same factor. Press Run Code and adjust the servings slider!
